margin

1、margin的auto属性值的作用

margin: auto; 和 margin: 0 auto; 能实现水平居中了。因为左右方向的auto值均分了可用空间，使得块元素得以在包含块内居中显示

margin-left: auto;

margin: 0 auto;

2、margin的值为percentage时的计算方法

水平书写模式（默认）下：margin的百分比计算值 = (percentage * 包含块的width)

非水平书写模式下：margin的百分比计算值 =(percentage * 包含块的height)

包含块（父级）width: 600px;

设定值margin: 10% 5%

计算值margin: 60px 30px

3、margin与relative定位的区别

margin-bottom 是增加自己与别人的距离

position: relative; bottom 是相对于自己的位置发生的偏移量

4、避免 margin 折叠的方法是：

  • 浮动
  • 非块元素
  • 绝对定位
  • overflow
  • 设border

5、负边距的应用：

5.1、列表边缘border或padding隐藏

  • li 的父级ul 设置 margin-bottom: -1px;
  • 可以隐藏掉最后一个 li 的 border-bottom
  • 同理可以隐藏最右边一个li 的 border-right
  • 负边距也适用于隐藏不想出现的 padding

5.2、伪等高布局

为需要登高布局的两个兄弟块设定：

    margin-bottom:-999px;
    padding-bottom:999px;
主内容栏
占位内容
侧边栏

5.3、悬挂

内容样式：padding-left: 3em;
悬挂样式：margin-left: -3em;

悬挂：内容——看起来很简单，没有浮动，没有绝对定位，没有其它重布局，甚至 HTML 也可以更简单，因为无需对后面那长段做任何处理，所以不需要再加包裹。这样的轻量方式，在一定时候还是有使用价值的，不是么？
